# ClinicFlow — Protótipo funcional do sistema clínico

Protótipo visual e funcional inspirado nas referências enviadas, com identidade própria e dados fictícios.

## Como executar

1. Extraia o arquivo ZIP.
2. Abra `index.html` no navegador.

Não exige instalação nem servidor nesta versão.

## Recursos incluídos

- Agenda diária com horários de 30 em 30 minutos.
- Calendário lateral navegável.
- Pesquisa de agendamentos.
- Cadastro e edição de agendamentos.
- Menu de status do atendimento.
- Lista e pesquisa de pacientes.
- Prontuário do paciente com resumo e histórico de consultas.
- Tela de nova consulta com abas: evolução, anamnese, exame físico, cirurgia, imprimíveis e arquivos.
- Cadastro completo de paciente com abas: dados gerais, outros e observações.
- Impressão de ficha do paciente.
- Persistência local com `localStorage`.

## Observação

Esta é uma versão front-end estática. Para produção, o ideal é migrar para uma arquitetura com:

- autenticação;
- banco de dados;
- API;
- controle de permissões;
- backup;
- LGPD;
- anexos reais em servidor.
