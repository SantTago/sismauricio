# Mauricio.Sis — versão final ajustada

Sistema clínico estático com agenda, pacientes, prontuário e dados salvos no navegador local.

## Ajustes finais incluídos

- Visual desktop mais compacto, simulando zoom menor.
- Menu de status reposicionado para não cortar no canto da tela.
- Status do paciente com cores diferentes.
- Responsividade mobile com menu lateral funcionando.
- Painel ⚙ com baixar, enviar/importar e apagar dados locais.
- Cadastro simplificado com os campos essenciais.
- Agenda com procedimento e tipo de consulta.

## Como executar

Abra `index.html` no navegador.
