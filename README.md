# Mauricio.Sis — versão atualizada

Sistema clínico estático com agenda, pacientes, prontuário, financeiro e dados salvos no navegador local.

## Ajustes incluídos nesta versão

- Procedimento e Tipo da agenda agora ficaram mais bonitos, em seletores individuais.
- Ao clicar, abre apenas a opção daquele campo, sem alterar o outro.
- Número do atendimento continua editável direto na agenda.
- Novo item **Financeiro** no menu lateral.
- Nova tela Financeiro com resumo do dia, registro de recebimento, tabela financeira, relatório mensal e controle de parcelamentos.

## Como executar

Abra `index.html` no navegador.

Observação: se o navegador já tiver dados antigos salvos, use o painel ⚙ para apagar os dados locais ou fazer novo backup/importação conforme necessário.
