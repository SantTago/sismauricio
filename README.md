# Mauricio.Sis — versão responsiva atualizada

Sistema clínico estático com agenda, pacientes, prontuário, financeiro e dados salvos no navegador local.

## Ajustes incluídos nesta versão

- Todas as áreas foram reforçadas para funcionar melhor no celular, tablet e computador.
- No celular, as opções da agenda não são mais ocultadas: Hora, Paciente, Procedimento, Tipo, Status e Ações aparecem em cartões responsivos, sem rolagem lateral.
- Menu lateral mobile com todas as opções visíveis ao abrir pelo botão ☰.
- Calendário, lembretes, aniversariantes, botão “Ver o dia de hoje” e alternância Dia/Semana permanecem disponíveis no celular.
- Agenda com campo **Horário livre**, permitindo agendar horários fora da grade padrão, como `08:05`.
- Financeiro redesenhado com cards por mês, cores por forma de pagamento e gráfico por Pix, Espécie, Crédito e Débito.
- Financeiro separado por mês: ao mudar o mês, os totais do novo mês começam zerados, mas os meses anteriores continuam acessíveis pelos botões do relatório mensal.
- Tabela financeira passa a mostrar os lançamentos apenas do mês selecionado.

## Como executar

Abra `index.html` no navegador.

Observação: se o navegador já tiver dados antigos salvos, use o painel ⚙ para apagar os dados locais ou fazer backup/importação conforme necessário.

## Ajuste responsivo final

- Removida a rolagem lateral no computador, tablet e celular.
- Agenda no celular passa a abrir em cartões com Hora, Paciente, Procedimento, Tipo, Status e Ações visíveis.
- Tabelas de Pacientes e Financeiro viram cartões no celular, sem esconder botões.
- Campos e botões agora quebram linha e se adaptam ao espaço disponível sem cortar conteúdo.
